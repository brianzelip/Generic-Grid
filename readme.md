##Generic Grid
made by philhoyt with love

##Getting Started
After downloading Generic Grid you will want to add ```generic-grid.min.css``` to your project.

```
<link rel="stylesheet" href="css/generic-grid.min.css">
```

#Basic Usage
```
<div class="container">
	<div class="row">
		<div class="col-m-4">
			<img src="http://placehold.it/500x500" />
		</div>
		<div class="col-m-4">
			<img src="http://placehold.it/500x500" />
		</div>
		<div class="col-m-4">
			<img src="http://placehold.it/500x500" />
		</div>
	</div>
</div>
```
